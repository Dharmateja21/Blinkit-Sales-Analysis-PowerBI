# Blinkit-Sales-Analysis-PowerBI
This project is an interactive Power BI Report Built to analyse Blinkit sales Performaance. This report provides insights into sales, item categories, outlet performance, customner ratings and outlet locations

## Tools used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Key KPIs
- Total sale: 1.20m
- Average Sales: 140.99
- Number of Items: 9k
- Aaverage Rating: 3.92

## Report Features
- Sales by Outlet Type
- Sales by Outlet Size
- Sales by Outlet Location
- Item Type Analysis
- Fat Content Analysis
- Otlet Establishment Trend
